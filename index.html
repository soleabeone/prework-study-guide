HTML

  • The head element provides data about the webpage
  • The body element is what the user sees
  • HTML is the framework of the website or provides structure

CSS

  • Cascading Style Sheets provide character to the HTML framework
  • CSS can be inline, internal, or external
  • External CSS allows for consistancy across the website and separates files
  • Using * applies the style to all elements or default style

Git

  • Git is a repository allowing coders to work together
  • Coders create branches that are reviewed and committed to the main branch

JavaScript

  • JavaScript is the functioning aspect of a website
